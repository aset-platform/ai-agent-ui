# ai-agent-ui — Coding Conventions (Qwen Delegate Context)

## Stack
Python 3.12 | FastAPI | LangChain 1.x | Next.js 16 | React 19 | TypeScript

## Python Rules
- Line length: 79 chars (black + isort + flake8 enforced)
- Type hints: `X | None` not `Optional[X]` (PEP 604)
- Logging: `logging.getLogger(__name__)` — NEVER `print()`
- Exceptions: always `except SomeError` — NEVER bare `except:`
- No module-level mutable globals — state in class instances
- Imports order: stdlib → third-party → local (isort --profile black)
- Naming: snake_case functions/vars, PascalCase classes
- Async: `async def` for all FastAPI endpoint handlers
- Pydantic models for all API request/response schemas
- Tests: pytest, fixtures in conftest.py, happy path + 1 error path min

## Agent Pattern
- `BaseAgent.__init__` → `_setup()` → `_build_llm()`
- Attrs used in `_build_llm()` must be set before super().__init__() or have defaults
- `FallbackLLM.invoke()` takes `iteration=` kwarg
- NO `@traceable` on `FallbackLLM.invoke()` — breaks LangChain tool parsing
- TokenBudget: use `reserve()`/`release()` (atomic) not `can_afford()`/`record()`
- StockRepository: always via `_require_repo()` from `tools/_stock_shared.py`

## TypeScript/Frontend Rules
- `apiFetch()` not `fetch()` — auto-refreshes JWT
- `<Image />` not `<img>` — ESLint enforced
- `NEXT_PUBLIC_BACKEND_URL` = `http://localhost:8181` — never `127.0.0.1`
- ESLint config: `frontend/eslint.config.mjs`

## Git
- Branch off `dev` — never push to dev/qa/release/main directly
- Mock patching: patch at SOURCE module, not the importing module

## File Locations
- Backend entry: `backend/main.py` (port 8181)
- Frontend entry: `frontend/app/page.tsx` (port 3000)
- Config: `backend/config.py` | Paths: `backend/paths.py`
- Auth: `auth/` (JWT + RBAC + OAuth PKCE)
- Data layer: `stocks/` (Iceberg, 20 tables)
- Env files: `~/.ai-agent-ui/backend.env`, `frontend.env.local`
